# Car Dealership Website

## Overview
This is a modern, responsive car dealership website built with Next.js 16, React 19, and Tailwind CSS.

## Prerequisites
To run this project, you need to have **Node.js** installed on your system.
- Recommended version: 18.x or later.
- Download it from: [nodejs.org](https://nodejs.org/)

## Installation
1.  Open your terminal in this directory.
2.  Install dependencies:
    ```bash
    npm install
    ```

## Development
To start the development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build for Production
To build the application for production:
```bash
npm run build
```
To start the production server:
```bash
npm start
```

## Project Structure
- `app/`: Contains the application routes (Next.js App Router).
- `components/`: UI components and page sections.
- `lib/`: Utility functions and mock data.
- `public/`: Static assets (images, icons).

## Features
- **Inventory Listing**: Browse cars with details.
- **Financing Page**: Form with validation for pre-approval.
- **Contact Page**: Location and contact form.
- **Responsive Design**: Optimized for mobile and desktop.
